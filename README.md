## Requirements:
 - node.js
 - running [backend](https://github.com/RodionChachura/simple-blog-back) 



## Run locally
```bash
git clone https://github.com/RodionChachura/simple-blog-front
cd simple-blog-front
npm install
npm start
```

## Technologies
* React
* Redux
* Redux-Saga
